How to Use

To use this tool, simply drag and drop an image of a currency note into the upload area, or click to select an image file from your computer. The system will automatically detect the type of currency.

  • Supported formats: JPEG, PNG
  • Ensure the image is clear and well-lit
  • Wait for the model to load before uploading

About This Project

This project uses a machine learning model to detect and identify currency notes from images. Simply upload an image, and the system will analyze it to predict the type of currency. This tool is built using TensorFlow.js for real-time processing directly in your browser.
